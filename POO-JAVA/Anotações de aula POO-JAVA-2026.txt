	Conceitos basicos:

Classe: Espaço para definir atributos e métodos (comportamento) que objetos terão.
Objeto: É uma instância concreta de uma classe, ocupando espaço na memória e possuindo estado e comportamento.
Estado: Os valores do Atributo em um instante de tempo
Método: Ações dos objetos definidas na Classe
Atributo: Caracteristicas dos objetos definidas na Classe


Aula 2 - 23/02/26
	
	Construtores:
Definição: Meio para inicializar um objeto (definir o estado inicial).
Deve ser usado para garantir que o objeto seja criado em um estado válido.
Pode ser usado, também, para rotinas iniciais (inicialização de um banco de dados, por exemplo)
O construtor é chamado apenas uma vez através do operador new.
Em Java, o construtor possui o mesmo nome da classe.

	Enum:
Definição: É um tipo especial que representa um conjunto fixo e finito de constantes.
Record
Definição: É uma classe sem atributos que variam (Constantes).
É uma classe especial imutável usada para representar dados. 
Seus atributos são automaticamente private final por padrão, não possuem Setters e recebem valores apenas no construtor.

	Exercício:

-2% para Gold
-3% para platinum
-5% para black
Implemente a situação descrita e um teste simples

	Modificadores de acesso

1 - Acesso a atributo
Leitura: Permite a leitura de um valor na memória através de um atributo.
	int x = obj.atributoInt +1;
Escrita: Permite atualização de um valor na memória através de um atributo.
	obj.atributoInt = 7;
2 - Acesso a método
Chamada: obj.metodo();
membro = método + atributo

Objetivo: Acoplamento baixo/coesão alta

Acoplamento: "O quanto uma classe A sabe/modifica o estado de outra classe B"
Coesão: "O quanto a classe A sabe/modifica o próprio estado"

Encapsulamento: É o princípio de esconder os detalhes internos da classe, expondo apenas o necessário através de métodos públicos.
Private: Acesso liberado para a própria classe
Public: Acesso liberado para todas as classes
Default: Acesso liberado para o pacote
Protected: Acesso liberado para o pacote e subclasses
Record: Para atributo, escrita sempre fechada (imutabilidade) e leitura aberta.

	Métodos Getters e Setters:

Getters: Acesso de leitura a um atributo.
Setters: Liberam acesso de escrita a um atributo.

 
	Modificador Static:
O modificador static desacopla o membro em questão da instância, tornando-o um membro de classe. Membros static pertencem à classe e não aos objetos. São compartilhados por todas as instâncias.
Obs.: Os valores das Enums são estáticos.
Cria um ponto de acesso global para o membro.

	Relacionamento entre classes

Tem um (has-a): É quando uma classe B (Enum/Record) possui uma referência a uma outra classe C.


	Execicio:
	Um celular possui bateria. A bateria possui uma carga inteira(0 a 100). Toda bateria pode carregar 1 ponto ou descarregar 1 ponto.
	Um celular ligado pode: recarregar; mostrar uma mensagem de bem vindo e desligar. A mensagem gasta um ponto de bateria. 
	Um celular desligado pode ser ligado e isto lhe custa, também, um ponto de bateria. Implemente a situação descrita e faça testes.
